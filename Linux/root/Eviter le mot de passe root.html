Eviter le mot de passe admin

Vous utilisez Ubuntu et vous êtes la seule personne à utiliser l’ordinateur, chaque fois que vous lancez un outil système votre mot de passe administrateur vous est demandé et à force c’est assez rébarbatif mais il y a un moyen d’éviter ça.

Pour cela ouvrez un terminal (console) et tapez sudo visudo bien sur cette fois encore vous devrez taper votre mot de passe

Vous allez obtenir ça :

Editez la ligne :  %admin    ALL=(ALL) ALL
 


 

Et remplacez par : %admin ALL=(ALL) NOPASSWD: ALL


 

Utilisez le raccourci clavier Ctrl + o puis appuyez sur Entrée pour enregistrer les modifications puis Ctrl + x pour fermer

Maintenant le mot de passe admin ne vous sera plus demandé
