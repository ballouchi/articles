What Are Symbolic Links?

Symbolic links are basically advanced shortcuts. Create a symbolic link to an individual file or folder, and that link will appear to be the same as the file or folder to Windows‚Äîeven though it‚Äôs just a link pointing at the file or folder.

There are two type of symbolic links: Hard and soft. Soft symbolic links work similarly to a standard shortcut.
